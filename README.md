## About The Project


Infinity Loops Bot is a discord bot created with discord.py. The bot webscrapes looperman.com and picks a random loop to send to help inspire an artist/producer.

<p align="right">(<a href="#top">back to top</a>)</p>

### Built With

* Python

### Packages Used

* discord.py
* beautifulsoup4
